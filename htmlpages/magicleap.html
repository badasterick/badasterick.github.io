Magic Leap SDK


Overview:

The Magic Leap 1 is an AR device created to harmonize people and technology to create a better, more united world. The Magic Leap Software Development Kit (SDK) gives developers the ability to create enterprise applications utilizing the Magic Leap device (some examples in the above video).

Role:

My work focused on the integration of the Magic Leap SDK into Unreal Engine 4. This included all new features developed for the Magic Leap SDK for subsequent software releases. My job was to set up developer test projects that would be used to guarantee new features would be 100% working at the time of release. Additionally I was in charge of an automation test pipeline encompassing over 40 different features reviewed with every release.

Two of my favorite projects I developed were a checkers and darts game. These games showcased wall and surface recognition, 3D object interaction, multiple user interaction at a set coordinate point, and raycasting. Magic Leap created a great video series that utilizes many of the features I worked on. Below is a raycast video tutorial from that series which demonstrates one of the ways I made it so a player can pick up a checker or dart.
